# a2a-go

Generated Go bindings for the A2A gRPC protocol used by production-grade agentic systems on the Alis Build platform.

This repository is automatically generated by Alis Build from the A2A protocol buffer definition and publishes the Go types and gRPC stubs needed to build A2A-compliant clients and services.

## Purpose

`a2a-go` is intended for builders on Alis Build who need strongly typed Go bindings for the Agent-to-Agent protocol. The generated package exposes:

- Protocol message types for tasks, messages, artifacts, agent cards, authentication, push notification configuration, and security schemes
- gRPC client and server interfaces for the A2A service
- Generated method metadata used by downstream tooling

The module path is:

```text
go.alis.build/a2a
```

The generated Go package is:

```text
go.alis.build/a2a/lf/a2a/v1
```

## Basic Usage

### Create a client

```go
package main

import (
	"context"

	a2av1 "go.alis.build/a2a/lf/a2a/v1"
	"google.golang.org/grpc"
	"google.golang.org/grpc/credentials/insecure"
)

func main() {
	conn, err := grpc.NewClient(
		"localhost:8080",
		grpc.WithTransportCredentials(insecure.NewCredentials()),
	)
	if err != nil {
		panic(err)
	}
	defer conn.Close()

	client := a2av1.NewA2AServiceClient(conn)

	_, err = client.SendMessage(context.Background(), &a2av1.SendMessageRequest{
		Message: &a2av1.Message{
			MessageId: "msg-1",
			Role:      a2av1.Role_ROLE_USER,
			Parts: []*a2av1.Part{
				{
					Content:   &a2av1.Part_Text{Text: "Summarize the latest task status."},
					MediaType: "text/plain",
				},
			},
		},
	})
	if err != nil {
		panic(err)
	}
}
```

### Implement a server

```go
package main

import (
	"context"

	a2av1 "go.alis.build/a2a/lf/a2a/v1"
)

type server struct {
	a2av1.UnimplementedA2AServiceServer
}

func (s *server) SendMessage(ctx context.Context, req *a2av1.SendMessageRequest) (*a2av1.SendMessageResponse, error) {
	return &a2av1.SendMessageResponse{
		Result: &a2av1.SendMessageResponse_Message{
			Message: &a2av1.Message{
				MessageId: "agent-1",
				Role:      a2av1.Role_ROLE_AGENT,
				Parts: []*a2av1.Part{
					{
						Content:   &a2av1.Part_Text{Text: "Request received."},
						MediaType: "text/plain",
					},
				},
			},
		},
	}, nil
}
```

## Intended Use on Alis Build

This package is meant to be consumed by builders implementing production-grade A2A systems on Alis Build, including:

- agent runtimes exposing A2A-compliant gRPC services
- control plane and orchestration services that call remote A2A agents
- platform adapters that translate internal workflows into standard A2A task, message, and artifact structures
- integration layers that need a stable generated contract rather than handwritten protocol types

Because this repository is generated, manual edits to the Go bindings should be avoided. Changes should come from the source proto and generation pipeline on the Alis Build platform.

## Generation Notes

The generated files include `DO NOT EDIT` headers and are intended to be reproducible outputs of the Alis Build code generation pipeline. Consumers should treat this repository as a published contract package, not as the source of truth for the specification itself.

The source of truth for protocol semantics remains the official A2A specification and the Alis Build generation workflow that tracks it.
